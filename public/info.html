Users

Get

  • Get / = root info page
  • Get /users = all users

Post

  • Post /user = find a specific user in the database => body: {email, password}
  • Post /newUser = post a user in th database => body: {email, password, department}

PUT

  • Put /user = changed the password => password: req.body.password, setPassword: req.body.setPassword;
  • Put /question = update the question from a specific user => body: {id, setQuestions}

Delete

  • Delete /users/:id = Delete user =>id
